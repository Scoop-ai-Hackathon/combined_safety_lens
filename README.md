# 🛡️ Safety Lens (세이프티 렌즈)

## 📌 프로젝트 소개
**Safety Lens**는 산업 현장의 안전 사각지대를 해소하고 작업자의 생명을 보호하기 위해 개발된 AI 기반 지능형 산업 안전 솔루션입니다. 본 프로젝트는 **Scoop AI 해커톤**에서 24시간이라는 제한된 시간 내에 다국적 팀원들과의 긴밀한 협업을 통해 완성되었으며, 뛰어난 기획력과 아키텍처 설계 역량을 인정받아 **우수상**을 수상하였습니다.

## 📝 프로젝트 설명
산업 현장의 카메라 영상을 실시간으로 분석하여 작업자의 안전 장비 착용 여부를 감지하고, 위험 상황을 즉각적으로 판별합니다. 제한된 환경에서도 안정적으로 동작할 수 있도록 아키텍처 관점에서 확장성과 안정성을 깊이 고려하여 설계되었습니다. 이를 통해 돌발적인 위험 상황에서도 지연 없는 빠른 알림을 전송하여 산업 재해를 사전 예방하는 데 초점을 맞췄습니다.

## 🛠 기술 스택 (Tech Stack)
*   **Back-end:** Java / Spring Boot, Flask
*   **AI / ML:** Multi-modal LLM (Qwen, GPT), Computer Vision
*   **Database:** MySQL
*   **Infra & DevOps:** AWS EC2, GitHub Actions (CI/CD 자동화)

## 🏗 시스템 아키텍처
Safety Lens의 핵심은 빠르고 안정적인 데이터 처리를 위한 **투트랙 아키텍처(Dual-track Architecture)**에 있습니다.
*   **Track 1 (실시간 처리):** 영상 프레임 내 객체를 실시간으로 1차 탐지하여 시스템 지연(Latency)을 최소화합니다.
*   **Track 2 (심층 분석):** 복잡한 상황 판단이 필요한 경우, Multi-modal LLM과 AI 서버를 통해 정밀 분석을 수행합니다.

이러한 분리 구조를 통해 실시간 알림의 신속성을 보장하는 동시에, 분석의 정확성을 높이고 트래픽 병목 현상을 방지했습니다.
